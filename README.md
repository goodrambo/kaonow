# KaoNow 考NOW

> 現在就開始，一次考過

台灣最好用的考試練習平台，整合證照、英文檢定、高普考、學測等各類考試題庫，提供講義、題庫與時事整合的一站式備考體驗。

## 功能特色

- 首頁導引 — 清楚說明平台能做什麼、用戶下一步怎麼做
- 考試目錄 — 依類別快速篩選（技術士、政府認證、英檢、高普考、升學、金融商業）
- 考試詳情頁 — 概覽 / 講義 / 題庫 / 時事新聞，四大分頁一次看全
- 章節練習 — 按考科章節逐步學習
- 隨機測驗 — 隨機抽題模擬真實情境
- 模擬考試 — 完整模擬正式考試
- 弱點加強 — 自動分析錯題，重點突破
- 個人中心 — 累計題數、正確率、連勝紀錄、最近練習
- 本地儲存 — 學習紀錄保存在瀏覽器（localStorage），免註冊
- 響應式設計 — 手機、平板、桌機均適用

## 技術

純前端靜態網站，使用原生 HTML + CSS + JavaScript，託管於 GitHub Pages。

- 單頁應用（SPA），Hash 路由
- 無框架依賴，載入即用
- 資料層與視圖分離，新增考試只需在 `exams` 與 `questionBanks` 加資料
- 使用者進度透過 `localStorage` 保存

## 本地預覽

```bash
# 用 Python 內建 server
python3 -m http.server 8000
# 或 Node
npx serve .
```

然後開啟 <http://localhost:8000>。
